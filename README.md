# JavaScript Fundamentals Exercises

This repository contains a collection of beginner-friendly JavaScript exercises focused on practicing core concepts such as DOM manipulation, events, and basic application logic.

The project is structured as a set of independent exercises, each with its own HTML and JavaScript file, and a main landing page (`index.html`) that links to all of them.

---

## 🚀 Live Demo

👉 https://mariafernandamamone.github.io/Javascript-Fundamentals-Exercises/

---

## 📁 Project Structure


Each exercise is separated into:
- One HTML file (structure and UI)
- One JavaScript file (logic and behavior)

---

## 🧠 Concepts Covered

- DOM manipulation
- Event handling
- Dynamic content rendering
- Form handling and validation
- Timers (`setInterval`, `clearInterval`)
- String manipulation
- Real-time updates
- LocalStorage (data persistence)

---

## 📋 Exercises

### 1. Background Color Changer
Changes the page background color to a random color when a button is clicked.

### 2. Click Counter
Counts how many times a button is clicked and updates the value in real time.

### 3. Dynamic List
Allows users to add and remove items from a list dynamically.

### 4. Real-Time Search Filter
Filters a predefined list based on user input as they type.

### 5. Simple Calculator
Performs basic arithmetic operations (add, subtract, multiply, divide) with input validation.

### 6. Timer (Start / Pause / Reset)
Implements a timer with start, pause, and reset functionality.

### 7. Random Password Generator
Generates secure random passwords based on user-defined length.

### 8. Word and Character Counter
Displays the number of words and characters in real time as the user types.

### 9. To-Do List with LocalStorage
A task manager that persists data using localStorage, including completion tracking.
