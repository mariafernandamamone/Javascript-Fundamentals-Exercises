# JavaScript Fundamentals Exercises

This repository contains a collection of beginner-friendly JavaScript exercises focused on practicing core concepts such as DOM manipulation, events, and basic application logic.

The project is structured as a set of independent exercises, each with its own HTML and JavaScript file, and a main landing page (`index.html`) that links to all of them.

---

## 🚀 Live Demo

This project is intended to be deployed using GitHub Pages.
